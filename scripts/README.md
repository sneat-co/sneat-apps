# Scripts for working with Sneat apps

## deploy.ps1

Builds & deploys apps with respective website.
To run pass app name as 1st parameter as:

```powershell
deploy.ps1 datatug
```

## list-uncovered-lines.mjs

Lists projects with the largest number of uncovered lines in descending order.

### Usage

```bash
# Analyze existing coverage data
npm run coverage:analyze
# or
node scripts/list-uncovered-lines.mjs

# Run tests with coverage first, then analyze
npm run coverage:analyze:run
# or
node scripts/list-uncovered-lines.mjs --run-tests
```
